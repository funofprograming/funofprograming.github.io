open override fun getEntriesSubMap(fromEventInstant: Instant, fromInclusive: Boolean, toEventInstant: Instant, toInclusive: Boolean): NavigableMap<Instant, Collection<TimeseriesEntry<E>>>

Get entries sub map based on from and to instant time period. Note that changes to this sub map will NOT alter the Timeseries

Return

NavigableMap<Instant, Collection<TimeseriesEntry>> of entries between fromEventInstant and toEventInstant

Parameters

fromEventInstant

From instant

fromInclusive

Whether to include fromEventInstant in the search. If true then returned sub map entries will all have instant >= fromEventInstant else instant > fromEventInstant.

toEventInstant

toInclusive

Whether to include toEventInstant in the search. If true then returned sub map entries will all have instant <= toEventInstant else instant < toEventInstant.
