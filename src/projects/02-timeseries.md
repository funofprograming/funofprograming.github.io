## Overview

Timeseries is a simple kotlin based library that provides a data structure to manage timeseries data.

## Usage

### Maven dependency

```xml
<dependency>
    <groupId>io.github.funofprograming</groupId>
    <artifactId>timeseries</artifactId>
    <version>LATEST</version>
</dependency>
```
